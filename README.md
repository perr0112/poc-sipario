# 🧪 Sipario - Proof of Concept (PoC)

This is a standalone repository designed to demonstrate the features of the [**@perr0112/sipario**](https://www.npmjs.com/package/@perr0112/sipario) package.

## 🎯 PoC Objectives

This project implements a complete SPA experience using static files to validate:

- **GSAP Orchestration:** Seamless page transitions using the `leave` and `transition` hooks.
- **Async Lifecycle:** Proper `init()` and `destroy()` execution for page-specific logic.
- **SEO & Metadata Sync:** Real-time updates of document titles and meta tags.

## 🚀 Getting Started

1. **Clone the repository:**
```bash
git clone [https://github.com/YOUR_USERNAME/sipario-poc.git](https://github.com/YOUR_USERNAME/sipario-poc.git)
cd sipario-poc
```
Install dependencies:

```
npm install
```

Run the development server:

```
npm run dev
```

Powered by Sipario — A lightweight SPA-like transition manager.
